### Start Funnel (run this to enable public access)
```bash
sudo tailscale funnel --https=443 --set-path=/ http://127.0.0.1:18789
```

### Check Status
```bash
# Tailscale status
tailscale status

# OpenClaw gateway status  
openclaw gateway status

# Pending pairings
openclaw devices list
```
